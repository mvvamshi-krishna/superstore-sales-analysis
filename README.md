# Retail Superstore Sales Analysis

## Overview
This project performs exploratory data analysis (EDA) on a sample retail superstore dataset to uncover trends in sales, profitability, customer behavior, and regional performance.

## Dataset
- Sample Superstore Dataset (educational dataset)
- 9,994 transactions
- Time period: 2014–2017

## Tools & Technologies
- Python
- pandas
- matplotlib
- numpy

## Key Analyses
- Data cleaning and feature engineering
- Regional sales and profit analysis
- Category and sub-category performance
- Customer segment analysis
- Time-series and seasonal trend analysis
- KPI computation (revenue, profit, margin, AOV)

## Key Insights
- Technology category has the highest profit margins
- West region leads in total sales and profitability
- Strong seasonal sales peaks in Q4 (November–December)
- Consumer segment contributes the highest order volume

## Business Recommendations
- Focus marketing and inventory planning on high-performing regions
- Expand high-margin technology product lines
- Optimize inventory and logistics for Q4 seasonal demand
- Re-evaluate discount strategies for low-margin categories

## Files in this Repository
- `superstore_analysis.ipynb` – Full analysis and visualizations
- `Sample-Superstore.csv` – Dataset used for analysis

## Note
This project uses a publicly available sample dataset for educational and analytical purposes.
